🌱 Solar Powered Smart Irrigation System

📌 Overview

The Solar Powered Smart Irrigation System is designed to automate irrigation using renewable energy and smart sensing technology. It helps conserve water, reduce manual effort, and improve agricultural efficiency by supplying water only when required.

🎯 Objectives
Automate irrigation using soil moisture sensing
Utilize solar energy for sustainable operation
Minimize water wastage
Reduce human intervention in farming
⚙️ Features
🌞 Solar-powered system
💧 Automatic irrigation based on soil moisture
🔋 Energy-efficient and eco-friendly
🤖 Low maintenance and easy to use
🛠️ Components Used
Solar Panel
Soil Moisture Sensor
Microcontroller (e.g., Arduino)
Water Pump
Relay Module
Connecting Wires
🔧 Working Principle

The soil moisture sensor continuously monitors the moisture level in the soil.

If moisture is low, the system automatically turns ON the water pump
If moisture is sufficient, the pump turns OFF

The entire system is powered using solar energy, making it cost-effective and sustainable.
